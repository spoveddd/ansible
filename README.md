# spoveddd/ansible

Этот репозиторий содержит Ansible роли и плейбуки, которые я использую для автоматизации различных задач. Он включает в себя примеры настройки серверов, приложений и сетевой инфраструктуры.

## Содержание
- Роли Ansible для различных сервисов
- Плейбуки для настройки серверов и приложений
